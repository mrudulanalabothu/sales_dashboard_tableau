# Task 2: Sales Dashboard in Tableau

## 🎯 Objective
To visualize and analyze sales and profit data using Tableau through multiple sheets and an interactive dashboard.

## 🪜 Steps Performed
- Imported the dataset into Tableau.
- Created individual sheets for different visualizations.
- Designed charts to show category, region, and profit insights.
- Combined all sheets into a single interactive dashboard.
- Added filters and actions for dynamic exploration.
- Formatted layout, titles, and colors for clarity.

## 📊 Sheets Included
- **Sheet 1:** Sales by Category (Bar Chart)
- **Sheet 2:** Sales by Region (Line Chart)
- **Sheet 3:** Sales by Category (Pie Chart)
- **Sheet 4:** Sales Heat Map
- **Sheet 5:** Profit Distribution (Box Plot)
- **Sheet 6:** Dashboard – combines all sheets into one interactive view with filters and actions.

## 🧰 Tools Used
- Tableau Desktop  
- Excel (for initial data cleaning)

Successfully created a Tableau dashboard that visualizes sales and profit performance across categories, regions, and time, enabling clear business insights.
